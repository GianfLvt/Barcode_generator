## Come usare

1. Apri `index.html` nel tuo browser
2. (Opzionale) Inserisci il nome del prodotto
3. Inserisci il codice che vuoi trasformare in barcode
4. Clicca su "Genera Barcode"
5. Puoi stampare o scaricare il barcode generato

## Hosting su GitHub Pages
